# kwilt
Quilt pattern creator

## Params
* Block sizes
  * 4, 6, 8, 9, 10 12 inch
* Tiles (edges can be 1-12", either 1, 2, 3, 4, 6, 8, 9, 10, 12 [1, or multiple of 3, or even])
  * square
  * right triangles
  * parallelogram (rect perimeter but skewed)
  * rectangles 2:1
 
## UI
  * Essentially a drawing app that tiles in squares, rectangles or triangles
    * select shape type
    * +/- to grow shrink
  * Save/load files
  * File edit history
  * Auto-save
  * Undo/Redo
  * Generates a cut-list of shapes
